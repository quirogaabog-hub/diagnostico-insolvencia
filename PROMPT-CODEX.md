# Instrucciones para Codex

Construye una landing page interactiva para Quiroga Abogados llamada:

“Diagnóstico de Riesgo de Insolvencia Empresarial”.

## Stack sugerido
- Next.js
- TypeScript
- Tailwind CSS

## Requisitos
Crear una aplicación web responsive con:

1. Página inicial elegante.
2. Test de 10 preguntas.
3. Tres opciones por pregunta.
4. Cálculo automático del puntaje.
5. Resultado según clasificación.
6. Formulario de contacto.
7. Botón de agendamiento.
8. Disclaimer legal.
9. Diseño moderno, sobrio y premium.

## Estilo visual
Firma legal boutique elegante y moderna.

Colores corporativos:
- Azul Magistral: #1D1F37
- Azul Corporativo: #243B73
- Oro Legal: #C9A869
- Beige Ejecutivo: #ECD7A8
- Marfil: #FFF1E1

## Funcionalidad
- El usuario debe poder responder todas las preguntas.
- No se debe mostrar resultado si falta alguna respuesta.
- Al finalizar, debe mostrar el resultado.
- El formulario debe aparecer antes o después del resultado.
- El código debe estar limpio, ordenado y comentado.
- Separar la lógica de preguntas, puntaje y resultado en archivos reutilizables.

## Resultado esperado
Una primera versión funcional lista para desplegar en Vercel.
