# Diagnóstico de Riesgo de Insolvencia Empresarial

Herramienta desarrollada por Quiroga Abogados para identificar señales tempranas de riesgo de insolvencia empresarial.

## Objetivo

Permitir que empresarios y gerentes evalúen el estado jurídico, financiero y gerencial de su empresa mediante un cuestionario estructurado.

## Resultado

La herramienta clasifica el riesgo en:

- Bajo
- Medio
- Alto
- Crítico
